# PhoneInfo-App
The Phone Info App is an App that is made using the Kivy Framework in Python. This app is used to 
Get Information about a particular phone number.

It provides information like:
1. Country Code
2. National Number
3. Timezone of the Number
4. Country
5. Service Provider
6. Validity
